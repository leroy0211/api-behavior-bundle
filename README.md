Skeleton project for new Symfony bundles.

## Installation

Add dependency:

```sh
composer require baxmusic/api-toolkit-bundle
```

Enable bundle:

Symfony 3

```php
$bundles[] = new BaxMusic\Bundle\ApiToolkit\BaxMusicApiToolkitBundle();
```

## Configuration

```yaml
# app/config/config.yml
baxmusic_api_toolkit:
```
